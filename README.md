# ai-resume-analyzer
A Generative AI project to analyze resumes against job descriptions and improve ATS compatibility using prompt engineering


## 🚀 Project Overview
AI Resume Analyzer is a Generative AI-powered tool that analyzes a resume against a job description and helps improve its ATS compatibility. It uses prompt engineering and LLM-based logic to:
- Compare resume text with job description
- Calculate a match percentage
- Identify key skills and missing keywords
- Provide clear optimization suggestions
- Suggest improved resume bullet points tailored to the role

This tool showcases practical prompt design and prompt-driven workflow logic, without relying on heavy coding.

---

## 🧠 Problem Statement
Most job seekers struggle with automated resume screening systems (ATS) due to missing keywords, unclear structure, and poor alignment with job requirements. This project helps bridge that gap by generating actionable feedback using Generative AI.

---

## 🛠️ Features
- 📝 Skill match evaluation against job description
- 🔍 Missing keyword detection
- 📈 ATS-focused feedback and improvement suggestions
- ✍️ Tailored bullet point recommendations
- 👇 Prompt logic that works with any resume & job description

---

## 📌 How It Works
1. **Input**: Provide resume text + job description
2. **Analyze**: The prompts compare and evaluate context
3. **Output**: Match %, required skills, suggestions, optimized bullets

This approach uses prompt engineering to produce business-ready output, demonstrating real Generative AI thinking. 1
